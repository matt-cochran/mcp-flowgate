Welcome. These docs cover everything you need to go from first install to production deployment.

**Getting Started** -- If you're new here, start with [What is mcp-flowgate?](/docs/introduction), then [install it](/docs/installation) and run through the [quick start](/docs/quick-start).

**Guides** -- Once you're up and running, the guides walk you through each feature: [discovery](/docs/guides/discovery), [governance](/docs/guides/governance), [workflows](/docs/guides/workflows), [connections](/docs/guides/connections), [deterministic chaining](/docs/guides/chaining), and more.

**Reference** -- Need to look something up? The reference section covers [configuration keys](/docs/reference/configuration), [MCP tools](/docs/reference/tools), [guards](/docs/reference/guards), [executors](/docs/reference/executors), [stores](/docs/reference/stores), and [audit events](/docs/reference/audit).

**Advanced** -- For embedding `mcp-flowgate-core` as a Rust library or understanding the [control architecture](/docs/advanced/architecture), head to the advanced section.

Source code and issues live on [GitHub](https://github.com/matt-cochran/mcp-flowgate).
